## 第一章 系统概述

### 1.1 系统简介

局域网文件共享系统是一套运行在本地局域网内的文件管理平台，面向企业、教育机构及团队协作场景，提供安全可控的文件存储、共享与协作服务。系统采用前后端分离架构，具备部署简单、使用便捷、安全可靠等特点。

本系统支持多人同时在线访问，用户可以通过浏览器随时随地访问和管理文件，实现文件的集中存储、权限控制和协作共享。所有数据存储在本地服务器，数据安全完全自主可控。

### 1.2 功能特性

本系统主要功能特性包括：

**文件管理功能**
- 支持多种格式文件的上传和下载
- 支持大文件分片上传，断点续传
- 支持文件在线预览（图片、PDF、文本、音视频等）
- 支持文件批量操作（下载、删除）
- 支持文件夹分级管理
- 支持文件搜索和多条件筛选
- 支持回收站功能，误删可恢复

**权限控制功能**
- 基于角色的访问控制（RBAC）
- 多级角色体系：超级管理员、管理员、普通用户、访客
- 文件级权限设置
- 操作审计日志

**协作共享功能**
- 文件分享链接生成
- 分享密码保护
- 分享有效期设置
- 好友系统
- 群组协作
- 即时聊天

**系统管理功能**
- 用户管理
- 部门管理
- 系统配置
- 数据备份与恢复

### 1.3 运行环境

**客户端环境**
- 操作系统：Windows 7及以上、macOS 10.12及以上、主流Linux发行版
- 浏览器：Chrome 80+、Firefox 75+、Edge 80+、Safari 13+
- 网络：支持局域网访问

**服务端环境**
- 操作系统：Windows Server 2012+、Ubuntu 18.04+、CentOS 7+
- Python：3.10及以上版本
- Node.js：18及以上版本
- 数据库：SQLite（开发环境）或 PostgreSQL（生产环境）

## 第二章 系统安装与配置

### 2.1 环境要求

在安装部署本系统之前，请确保服务器满足以下环境要求：

**硬件要求**
- CPU：双核及以上处理器
- 内存：4GB及以上
- 硬盘：根据文件存储需求配置，建议100GB以上

**软件要求**
- Python 3.10 或更高版本
- Node.js 18 或更高版本
- npm 包管理器

### 2.2 安装部署

**Windows环境一键部署**

步骤一：下载系统安装包

从指定渠道获取系统安装包，解压到目标目录。

步骤二：运行启动脚本

双击运行 start.bat 文件，系统将自动完成以下操作：
1. 检查 Python 和 Node.js 环境
2. 安装后端依赖包
3. 配置环境变量
4. 安装前端依赖包
5. 启动后端服务（端口8088）
6. 启动前端服务（端口3088）

步骤三：访问系统

打开浏览器，输入地址 http://localhost:3088 即可访问系统。

**手动部署**

后端部署步骤：

步骤一：进入后端目录
```
cd backend
```

步骤二：安装依赖
```
pip install -r requirements.txt
```

步骤三：配置环境变量
```
copy .env.example .env
```

步骤四：启动后端服务
```
uvicorn app.main:app --host 0.0.0.0 --port 8088
```

前端部署步骤：

步骤一：进入前端目录
```
cd frontend
```

步骤二：安装依赖
```
npm install
```

步骤三：启动开发服务器
```
npm run dev
```

### 2.3 系统配置

系统配置文件位于 backend/.env，主要配置项说明如下：

**应用配置**
- APP_NAME：应用名称，默认为"局域网文件共享系统"
- APP_ENV：运行环境，development（开发）或 production（生产）
- DEBUG：调试模式，true 或 false
- SECRET_KEY：安全密钥，生产环境务必修改为随机字符串

**服务配置**
- HOST：服务监听地址，默认 0.0.0.0
- PORT：服务端口，默认 8088

**数据库配置**
- DATABASE_URL：数据库连接字符串，默认使用 SQLite

**文件存储配置**
- MAX_UPLOAD_SIZE：最大上传文件大小，0表示不限制
- ALLOWED_EXTENSIONS：允许上传的文件扩展名

## 第三章 用户注册与登录

### 3.1 用户注册

首次使用系统需要注册账号，操作步骤如下：

步骤一：打开系统首页

在浏览器地址栏输入系统访问地址，打开登录页面。

步骤二：点击注册链接

在登录页面底部，点击"立即注册"链接，进入注册页面。

步骤三：填写注册信息

在注册页面填写以下信息：
- 用户名：4-20个字符，支持字母、数字、下划线
- 密码：6-20个字符，建议包含字母和数字
- 确认密码：再次输入密码
- 真实姓名：可选填写
- 邮箱：可选填写

步骤四：提交注册

点击"注册"按钮提交注册信息。注册成功后自动跳转到登录页面。

注意事项：
- 用户名不能与已注册用户重复
- 密码需要输入两次并保持一致
- 管理员可能需要审核后才能使用账号

### 3.2 用户登录

步骤一：打开登录页面

在浏览器地址栏输入系统访问地址，打开登录页面。

步骤二：输入账号信息

输入用户名和密码。

步骤三：点击登录

点击"登录"按钮，验证通过后进入系统主界面。

默认管理员账号：
- 用户名：admin
- 密码：admin123

安全提示：
- 首次登录后请及时修改默认密码
- 不要在公共场所保存登录状态
- 定期更换密码以保证账号安全

### 3.3 密码修改

步骤一：进入个人中心

点击右上角用户头像，选择"个人中心"进入个人设置页面。

步骤二：选择密码修改

在个人中心页面，找到"修改密码"选项。

步骤三：填写密码信息

输入当前密码和新密码，新密码需要输入两次确认。

步骤四：保存修改

点击"保存"按钮完成密码修改。

### 3.4 退出登录

点击右上角用户头像，选择"退出登录"选项，系统将清除登录状态并返回登录页面。

建议在离开电脑前退出登录，以保护账号安全。

## 第四章 文件管理

### 4.1 文件上传

系统支持单个文件和批量文件上传，操作步骤如下：

**单文件上传**

步骤一：点击上传按钮

在文件管理页面，点击工具栏的"上传文件"按钮。

步骤二：选择文件

在弹出的对话框中，可以通过以下方式选择文件：
- 点击上传区域选择文件
- 直接将文件拖拽到上传区域

步骤三：开始上传

点击"开始上传"按钮，系统开始上传文件。上传过程中会显示进度条。

步骤四：确认上传

上传完成后，文件列表会自动刷新显示新上传的文件。

**批量上传**

系统支持同时选择多个文件上传，操作方式与单文件上传相同，选择多个文件后点击"开始上传"即可。

**大文件上传**

对于大文件（超过50MB），系统会自动启用分片上传功能：
- 将文件分割成多个小块分别上传
- 支持断点续传，网络中断后可继续上传
- 上传完成后自动合并文件

注意事项：
- 上传文件名不能包含特殊字符
- 文件大小不能超过系统限制
- 确保网络连接稳定

### 4.2 文件下载

**单文件下载**

步骤一：选择文件

在文件列表中找到需要下载的文件。

步骤二：点击下载

方式一：在列表视图下，点击文件行的"下载"按钮
方式二：在卡片视图下，点击文件卡片下方的下载图标
方式三：右键点击文件，选择"下载"

步骤三：保存文件

浏览器会自动开始下载，根据浏览器设置保存文件到指定位置。

**批量下载**

步骤一：选择多个文件

在列表视图下，勾选需要下载的多个文件。

步骤二：点击批量下载

点击工具栏的"批量下载"按钮。

步骤三：保存压缩包

系统会将选中的文件打包成ZIP格式下载，解压后即可获取所有文件。

### 4.3 文件预览

系统支持多种文件格式的在线预览，无需下载即可查看文件内容。

**支持的预览格式**
- 图片：jpg、jpeg、png、gif、webp、bmp
- 文档：pdf
- 文本：txt、md、log、json、xml
- 视频：mp4、webm、mov
- 音频：mp3、wav、ogg

**预览操作**

步骤一：找到需要预览的文件

在文件列表中，可以预览的文件会显示"预览"按钮。

步骤二：点击预览

点击"预览"按钮，系统会在弹窗中显示文件内容。

步骤三：关闭预览

查看完成后点击关闭按钮或点击弹窗外部关闭预览窗口。

图片预览支持缩放查看，PDF文档支持翻页浏览，视频和音频支持播放控制。

### 4.4 文件搜索

系统提供快速搜索和高级搜索两种方式。

**快速搜索**

在文件列表页面，直接浏览查找需要的文件，可通过排序功能辅助查找。

**高级搜索**

步骤一：打开高级搜索

点击工具栏的"高级搜索"按钮。

步骤二：设置搜索条件

可设置以下搜索条件：
- 文件名：输入文件名关键词
- 文件类型：选择图片、文档、视频、音频、压缩包等
- 上传时间：选择日期范围
- 文件大小：选择大小范围

步骤三：执行搜索

点击"搜索"按钮，系统会筛选显示符合条件的文件。

步骤四：重置搜索

点击"重置"按钮清空所有搜索条件，显示全部文件。

### 4.5 文件重命名

步骤一：选择文件

找到需要重命名的文件，点击操作菜单中的"重命名"选项，或右键选择"重命名"。

步骤二：输入新名称

在弹出的对话框中输入新的文件名。

步骤三：确认修改

点击"确定"按钮保存新名称。

注意事项：
- 文件名不能为空
- 不能与同目录下其他文件重名
- 建议保留文件扩展名

### 4.6 文件移动

步骤一：选择文件

找到需要移动的文件，点击操作菜单中的"移动到"选项。

步骤二：选择目标文件夹

在弹出的文件夹树形结构中，选择目标文件夹。

步骤三：确认移动

点击"确定"按钮，文件将被移动到目标文件夹。

### 4.7 文件复制

步骤一：选择文件

找到需要复制的文件，点击操作菜单中的"复制到"选项。

步骤二：选择目标文件夹

在弹出的文件夹树形结构中，选择目标文件夹。

步骤三：确认复制

点击"确定"按钮，文件将被复制到目标文件夹。

复制操作会在目标位置创建文件副本，原文件不受影响。

### 4.8 文件删除

**删除单个文件**

步骤一：选择文件

找到需要删除的文件，点击操作菜单中的"删除"选项，或右键选择"删除"。

步骤二：确认删除

在确认对话框中点击"确定"，文件将被移入回收站。

**批量删除**

步骤一：选择多个文件

在列表视图下，勾选需要删除的多个文件。

步骤二：点击批量删除

点击工具栏的"删除"按钮。

步骤三：确认删除

在确认对话框中点击"确定"，选中的文件将被移入回收站。

注意事项：
- 删除的文件会先进入回收站，可以恢复
- 回收站中的文件可以彻底删除（管理员权限）
- 彻底删除后无法恢复

### 4.9 回收站管理

回收站用于临时存放已删除的文件，用户可以恢复或彻底删除这些文件。

**打开回收站**

点击工具栏的"回收站"按钮，打开回收站对话框。

**恢复文件**

在回收站中找到需要恢复的文件，点击"恢复"按钮，文件将恢复到原来的位置。

**彻底删除**

管理员用户可以点击"彻底删除"按钮永久删除文件，此操作不可恢复。

**清空回收站**

管理员用户可以点击"清空回收站"按钮一次性删除回收站中的所有文件。

## 第五章 文件夹管理

### 5.1 新建文件夹

步骤一：点击新建按钮

在文件管理页面，点击工具栏的"新建文件夹"按钮。

步骤二：输入文件夹名称

在弹出的对话框中输入文件夹名称。

步骤三：确认创建

点击"确定"按钮完成文件夹创建。

注意事项：
- 文件夹名称不能为空
- 同一目录下不能存在同名文件夹
- 文件夹名称建议使用有意义的描述

### 5.2 文件夹重命名

步骤一：选择文件夹

找到需要重命名的文件夹，右键点击选择"重命名"。

步骤二：输入新名称

在弹出的对话框中输入新的文件夹名称。

步骤三：确认修改

点击"确定"按钮保存新名称。

### 5.3 文件夹删除

步骤一：选择文件夹

找到需要删除的文件夹，右键点击选择"删除"，或在列表视图下勾选后点击删除按钮。

步骤二：确认删除

在确认对话框中点击"确定"。

注意事项：
- 删除文件夹会同时删除其中的所有文件和子文件夹
- 删除后的文件夹会进入回收站

### 5.4 文件夹导航

**进入文件夹**

在文件列表中双击文件夹，或单击文件夹名称，即可进入该文件夹。

**返回上层**

点击工具栏的"返回上层"按钮，或点击面包屑导航中的上级目录名称。

**面包屑导航**

页面顶部显示当前所在的目录路径（面包屑导航），点击任意目录名称可直接跳转到该位置。

**根目录**

点击面包屑导航最左侧的"根目录"可返回最顶层目录。

## 第六章 文件分享

### 6.1 创建分享

文件分享功能允许用户生成分享链接，让其他人无需登录即可访问文件。

步骤一：选择文件

在文件列表中找到需要分享的文件，点击操作菜单中的"分享"选项。

步骤二：设置分享参数

在分享对话框中设置以下参数：
- 访问密码：可选，设置后访问者需要输入密码
- 有效期：可选择1小时、6小时、24小时、3天、7天或永不过期
- 下载次数限制：设置最大下载次数，0表示不限制

步骤三：创建分享

点击"创建分享"按钮，系统生成分享链接。

步骤四：复制分享链接

分享创建成功后，在结果对话框中复制分享链接，可通过即时通讯工具发送给他人。

分享链接格式示例：http://服务器地址:端口/share/分享码

### 6.2 访问分享

步骤一：打开分享链接

在浏览器中打开收到的分享链接。

步骤二：输入密码

如果分享设置了访问密码，需要输入正确的密码才能访问。

步骤三：查看或下载文件

访问成功后可以在线预览或下载分享的文件。

注意事项：
- 分享链接可能已过期或被取消
- 下载次数可能已达上限
- 请勿分享敏感文件

### 6.3 管理分享

**查看我的分享**

在左侧菜单点击"我的分享"，可以查看所有已创建的分享记录。

分享列表显示以下信息：
- 分享文件名
- 分享码
- 创建时间
- 过期时间
- 下载次数
- 状态（有效/已过期）

**取消分享**

找到需要取消的分享记录，点击"删除"按钮，分享链接将失效。

**复制分享链接**

找到需要的分享记录，点击"复制链接"按钮重新获取分享链接。

## 第七章 空间管理

### 7.1 空间类型

系统提供三种类型的文件存储空间：

**个人空间**
- 每个用户独立的私有存储空间
- 仅用户本人可访问
- 用于存储个人文件

**公共空间**
- 所有用户可访问
- 根据权限进行文件管理
- 用于存储共享资源

**群组空间**
- 群组成员可访问
- 根据群组角色管理权限
- 用于团队协作

### 7.2 个人空间

个人空间是用户登录后的默认工作空间，用于管理个人文件。

**访问个人空间**

登录系统后默认进入个人空间的文件管理页面。

**个人空间功能**

个人空间支持所有文件管理功能：
- 文件上传下载
- 文件夹创建管理
- 文件分享
- 文件搜索

### 7.3 公共空间

公共空间用于存储需要多人共享的文件资源。

**访问公共空间**

在左侧导航菜单中点击"公共空间"进入。

**公共空间权限**

用户对公共空间的操作权限由管理员设置：
- 查看权限：可以浏览和下载文件
- 上传权限：可以上传新文件
- 删除权限：可以删除文件
- 管理权限：可以管理文件夹和权限设置

### 7.4 群组空间

群组空间与群组关联，群组成员可以协作管理文件。

**访问群组空间**

方式一：在"我的群组"中选择群组，进入群组空间
方式二：直接在左侧菜单选择群组空间入口

**群组空间权限**

群组空间权限与群组角色相关：
- 群主：拥有全部权限
- 管理员：可以管理文件和成员
- 成员：可以查看、上传、下载
- 访客：仅可查看

## 第八章 群组功能

### 8.1 创建群组

步骤一：进入群组管理

在左侧菜单点击"群组管理"，进入群组列表页面。

步骤二：点击创建群组

点击"创建群组"按钮，进入创建页面。

步骤三：填写群组信息

输入以下信息：
- 群组名称：必填，建议使用有意义的名称
- 群组描述：可选，介绍群组用途
- 加入方式：选择公开加入或需要邀请

步骤四：保存群组

点击"创建"按钮完成群组创建。

创建成功后，系统会自动：
- 创建群组空间
- 生成邀请码（如开启）
- 将创建者设为群主

### 8.2 群组成员管理

**查看成员列表**

进入群组详情页面，切换到"成员"标签页查看所有成员。

成员信息包括：
- 成员名称
- 成员角色（群主/管理员/成员/访客）
- 加入时间

**邀请成员**

群主和管理员可以邀请新成员加入：
1. 在成员管理页面点击"邀请成员"
2. 选择邀请方式：
   - 通过用户名搜索邀请
   - 分享邀请码让用户自行加入
3. 发送邀请

**移除成员**

群主可以移除群组成员：
1. 在成员列表中找到需要移除的成员
2. 点击"移除"按钮
3. 确认移除操作

**修改成员角色**

群主可以修改成员角色：
1. 在成员列表中找到目标成员
2. 点击角色下拉框选择新角色
3. 保存修改

### 8.3 群组邀请

**获取邀请码**

群主和管理员可以获取群组邀请码：
1. 进入群组详情页面
2. 在基本信息中查看邀请码
3. 可以重新生成邀请码

**邀请链接**

邀请链接格式：http://服务器地址:端口/join/邀请码

将邀请链接发送给需要加入的用户，用户点击链接即可申请加入群组。

### 8.4 加入群组

**通过邀请码加入**

步骤一：获取邀请码或邀请链接

从群主或管理员处获取邀请码或邀请链接。

步骤二：打开加入页面

直接点击邀请链接，或在系统中输入邀请码。

步骤三：确认加入

查看群组信息，点击"加入群组"按钮。

步骤四：等待审核

如果群组需要审核，需要等待管理员批准。

**查看已加入的群组**

在左侧菜单"群组管理"中查看已加入的所有群组。

## 第十三章 常见问题

### Q1：忘记密码怎么办？

请联系系统管理员重置密码。管理员可以在用户管理页面为您重置密码。

### Q2：上传文件失败怎么办？

请检查以下项目：
1. 文件格式是否在允许范围内
2. 文件大小是否超过限制
3. 网络连接是否正常
4. 磁盘空间是否充足

### Q3：如何找回删除的文件？

删除的文件会进入回收站，可以在回收站中恢复。如果回收站中的文件已被清空，则无法恢复。

### Q4：分享链接打不开怎么办？

可能的原因：
1. 分享已过期
2. 分享已被创建者取消
3. 下载次数已达上限
4. 输入的密码不正确

### Q5：无法登录系统怎么办？

请检查以下项目：
1. 用户名和密码是否正确
2. 账号是否被禁用
3. 网络连接是否正常
4. 清除浏览器缓存后重试

### Q6：文件预览不显示怎么办？

可能的原因：
1. 文件格式不支持预览
2. 文件已损坏
3. 浏览器版本过低，建议升级浏览器

### Q7：如何修改用户名？

用户名创建后不可修改，如需更换用户名请联系管理员创建新账号。

### Q8：群组空间文件无法删除？

请检查您的群组角色权限，只有群主和管理员可以删除群组空间文件。
